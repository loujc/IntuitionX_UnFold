# 视频文件存放目录

## 📁 文件组织

将演示视频文件放到这个目录下，命名规则：

```
/videos
  ├── medicine.mp4        # 医药价格分析视频
  ├── finance.mp4         # 金融市场分析视频（预留）
  └── history.mp4         # 明朝历史讲解视频（预留）
```

## 🎯 如何添加新视频

1. 将视频文件重命名为对应的 Demo ID（如 `medicine.mp4`）
2. 放入此目录
3. 在 `src/config/demoRegistry.ts` 中确认路径配置正确
4. 启动开发服务器即可使用

## 📝 注意事项

- 视频格式建议：MP4 (H.264 编码)
- 文件大小建议：小于 100MB（演示用）
- 分辨率建议：1920x1080 或 1280x720
- 确保视频文件名与 `demoRegistry.ts` 中的 `videoPath` 一致
